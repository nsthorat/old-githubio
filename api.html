Parameters:
shape: ShapeMap[R]
An array of integers defining the output tensor shape.
dtype: "float32" | "int32" | "bool"
(Optional) The dtype of the buffer. Defaults to 'float32'.

# dl.reshape
Reshapes a tensor. Given a input tensor, returns a new tensor with the same values as the input tensor with shape `shape`. If one component of shape is the special value -1, the size of that dimension is computed so that the total size remains constant. In particular, a shape of [-1] flattens into 1-D. At most one component of shape can be -1. If shape is 1-D or higher, then the operation returns a tensor with shape shape filled with the values of tensor. In this case, the number of elements implied by shape must be the same as the number of elements in tensor.
Parameters:
x: Tensor<Rank>
A tensor.
shape: ShapeMap[R2]
An array of integers defining the output tensor shape.

# dl.tile
Construct an tensor by repeating it the number of times given by reps. This operation creates a new tensor by replicating `input` `reps` times. The output tensor's i'th dimension has `input.shape[i] * reps[i]` elements, and the values of `input` are replicated `reps[i]` times along the i'th dimension. For example, tiling `[a, b, c, d]` by `[2]` produces `[a, b, c, d, a, b, c, d]`.
Parameters:
x: T
The tensor to transpose.
reps: {}
Determines the number of replications per dimension.

# dl.norm
Computes the norm of scalar, vectors, and matrices. This function can compute several different vector norms (the 1-norm, the Euclidean or 2-norm, the inf-norm, and in general the p-norm for p > 0) and matrix norms (Frobenius, 1-norm, and inf-norm).
Parameters:
x: Tensor<Rank>
The input array.
ord: number | "euclidean" | "fro"
(Optional) Optional. Order of the norm. Supported norm types are following: ord norm for matrices norm for vectors ------------------------------------------------------- 'euclidean' Frobenius norm 2-norm ‘fro’ Frobenius norm – Infinity max(sum(abs(x), axis=1)) max(abs(x)) -Infinity min(sum(abs(x), axis=1)) min(abs(x)) 1 max(sum(abs(x), axis=0)) sum(abs(x)) 2 - sum(abs(x)^2)^1/2*
axis: number | {}
(Optional) Optional. If axis is null (the default), the input is considered a vector and a single vector norm is computed over the entire set of values in the Tensor, i.e. norm(x, ord) is equivalent to norm(x.reshape([-1]), ord). If axis is a integer, the input is considered a batch of vectors, and axis determines the axis in x over which to compute vector norms. If axis is a 2-tuple of integer it is considered a batch of matrices and axis determines the axes in NDArray over which to compute a matrix norm.
keepDims: boolean
(Optional) Optional. If true, the norm have the same dimensionality as the input.

# dl.transpose
Transposes the array. Permutes the dimensions according to `perm`. The returned array's dimension `i` will correspond to the input dimension `perm[i]`. If `perm` is not given, it is set to `[n-1...0]`, where `n` is the rank of the input array. Hence by default, this operation performs a regular matrix transpose on 2-D input arrays.
Parameters:
x: Tensor<R>
